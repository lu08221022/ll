CSS语法基础

CSS（层叠样式表）用于描述HTML元素的显示方式。本练习展示CSS的不同应用方式，包括外部样式表、内部样式表和内嵌样式表。

在这个练习中，我们将学习如何使用不同的选择器来应用样式，包括元素选择器、类选择器、ID选择器和伪类选择器。理解这些选择器的优先级对于有效应用CSS至关重要。

选择符类型

基本选择器

  • 元素选择器（p, h1）
  • 类选择器（.abstract）
  • ID选择器（#CSSbase）

组合选择器

  • 后代选择器（div p）
  • 子元素选择器（ul > li）
  • 相邻兄弟选择器（h2 + p）

伪类选择器

  • :hover
  • :active
  • :nth-child()

样式应用方式

CSS可以通过三种主要方式应用到HTML文档中：

  • 外部样式表 - 使用link标签引入外部CSS文件
  • 内部样式表 - 在HTML文档中使用style标签定义样式
  • 内嵌样式 - 直接在HTML元素中使用style属性

CSS 核心概念：背景、盒模型、布局、动画

背景

"cascading" 的音标是 /kæˈskeɪdɪŋ/。在 HTML 文件里为 linear：英 [ˈlɪniə(r)] 美 [ˈlɪniər]

CSS背景属性用于定义元素的背景效果，包括背景颜色、背景图片、背景位置等。

常用背景属性：

  • background-color: 设置背景颜色
  • background-image: 设置背景图片
  • background-position: 设置背景图片位置
  • background-repeat: 设置背景图片是否重复

盒模型

盒模型是CSS布局的基础概念，每个元素都被视为一个矩形的盒子。

box 中的内容 1
box 中的内容 2

盒模型包含四个部分：

  • 内容(content) - 元素的实际内容
  • 内边距(padding) - 内容周围的透明区域
  • 边框(border) - 内边距周围的边框
  • 外边距(margin) - 边框周围的透明区域

动画

动画必须要有起始状态、结束状态、时长

盒模型内容 (悬停查看效果)

CSS动画使元素从一种样式逐渐变化为另一种样式，通过关键帧控制动画的中间步骤。

常用动画属性：

  • @keyframes - 定义动画序列
  • animation-name - 指定动画名称
  • animation-duration - 指定动画时长
  • animation-timing-function - 设置动画速度曲线
  • transition - 属性变化的过渡效果
